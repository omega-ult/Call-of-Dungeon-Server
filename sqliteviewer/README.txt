SQLite Viewer
Version 0.1

SQLite Viewer is an public domain open source tool used to browse SQLite database files.
You can find the source code and redistributable binaries at https://sourceforge.net/p/sqliteviewer/

Large Parts of the code are taken from SQLite Database Browser v2.2-a
SQLite Database Browser is an open source, public domain, freeware tool used to create, design and edit SQLite database files.

For more information on SQLite Database Browser please visit their site at: http://sqlitebrowser.sf.net
